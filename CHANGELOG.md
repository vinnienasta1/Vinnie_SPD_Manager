# Changelog

## [1.1.0-beta] — 2026-05-01

- Автообновление: фоновая проверка при старте, проверка в «О программе», кнопка «Проверка обновлений».
- Тихая установка обновления и отложенное напоминание, если пользователь отложил апдейт.
- Улучшена валидация обновления: проверка SHA-256 (если опубликован) и проверка цифровой подписи.
- В сборку включены WAV из папки `Metronome`, чтобы Click Track работал в установленной версии.

## [1.0.0] — 2026-05-01

Первый стабильный релиз **Vinnie SPD Manager** для Windows (x64, self-contained).

- Управление наборами и банками Roland SPD.
- Локальные данные и логи в `%LocalAppData%\VinnieSpdManager`.
- Установщик Inno Setup (`installer\build-setup.ps1`).
